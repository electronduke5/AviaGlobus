Для подключения к базе данных пользователю необходимо создать базу данных MS SQL
с названием AviaBD.

Далее пользователю необходимо открыть директорию в архиве как проект в любой среде разработки,
поддерживающей фреймворк ASP.NET.Core и язык C#. После чего пользователю необходимо
запустить проект с помощью IIS Express (в Visual Studio это обычная кнопка запуска).
